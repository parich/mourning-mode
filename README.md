# โหมดไว้อาลัย (Mourning Mode)

WordPress plugin เปลี่ยนหน้าเว็บเป็นโทนสีขาวดำ พร้อมแสดงโบว์ดำมุมขวาบน เพื่อน้อมรำลึกในพระมหากรุณาธิคุณ

## คุณสมบัติ

- เปิด/ปิดโหมดไว้อาลัยด้วยคลิกเดียว
- เลือกแสดง/ซ่อนโบว์ดำมุมขวาบนได้
- ปรับขนาดรูปโบว์ได้ (50-500 px)
- เลือกเปิด/ปิดโทนสีขาวดำแยกต่างหาก
- ปรับระดับความเข้มของ Grayscale ได้ (0-100%)
- รองรับ Responsive Design (Desktop, Tablet, Mobile)
- รองรับ WordPress Admin Bar
- แสดงผลเฉพาะหน้า Frontend (หน้า Admin แสดงปกติ)

## ความต้องการ

- WordPress 5.0 ขึ้นไป
- PHP 7.4 ขึ้นไป

## การติดตั้ง

1. ดาวน์โหลดหรือ clone โฟลเดอร์ plugin ไปไว้ที่ `wp-content/plugins/mourning-mode/`
2. เปิดใช้งาน plugin ผ่านหน้า WordPress Admin > Plugins
3. ไปที่ **Settings > โหมดไว้อาลัย** เพื่อตั้งค่า

## การตั้งค่า

| ตัวเลือก | คำอธิบาย |
|---|---|
| เปิดใช้งานโหมดไว้อาลัย | สวิตช์หลัก เปิด/ปิดทั้งระบบ |
| แสดงโบว์ไว้อาลัย | เลือกว่าจะแสดงรูปโบว์ดำมุมขวาบนหรือไม่ |
| ขนาดรูปโบว์ (px) | ปรับขนาดโบว์ 50-500 px (ค่าเริ่มต้น 150) |
| แสดงโทนสีขาวดำ | เลือกว่าจะเปลี่ยนเว็บเป็นโทนขาวดำหรือไม่ |
| ระดับความเข้ม (%) | ปรับระดับ Grayscale 0-100% (0% = สีปกติ, 100% = ขาวดำทั้งหมด) |

## โครงสร้างไฟล์

```
mourning-mode/
├── mourning-mode.php          # ไฟล์ plugin หลัก
├── black_ribbon_top_right.png # รูปโบว์ดำ (พื้นหลังโปร่งใส)
└── README.md
```
