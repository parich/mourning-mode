เปลี่ยนหน้าเว็บทั้งหมดเป็นโทนสีขาวดำ พร้อมแสดงโบว์ดำมุมขวาบน เพื่อน้อมรำลึกในพระมหากรุณาธิคุณ

== Description ==

Plugin นี้จะเปลี่ยนหน้าเว็บไซต์ทั้งหมดเป็นโทนสีขาวดำ (Grayscale) พร้อมแสดงโบว์ดำมุมขวาบน เพื่อแสดงความไว้อาลัย

คุณสมบัติ:
* เปลี่ยนทั้งเว็บไซต์เป็นโทนสีขาวดำด้วยการคลิกเดียว
* แสดงโบว์ดำมุมขวาบนของหน้าเว็บ
* รองรับทั้ง Desktop และ Mobile (ขนาดโบว์ปรับตามหน้าจอ)
* รองรับ WordPress Admin Bar
* ใช้งานง่าย เปิด-ปิด ได้ทันที

== Installation ==

1. ติดตั้ง plugin ผ่านหน้า WordPress Admin > Plugins > Add New > Upload Plugin
2. เลือกไฟล์ ZIP ของ plugin แล้วกด Install Now
3. กด Activate Plugin
4. ไปที่ Settings > โหมดไว้อาลัย
5. ติ๊กเลือก "เปิดใช้งานโหมดไว้อาลัย" แล้วกด Save Changes

หมายเหตุ: ต้องวางไฟล์รูป `black_ribbon_top_right.png` ในโฟลเดอร์ plugin เพื่อแสดงโบว์ดำ

== Frequently Asked Questions ==

= จะเปิดหรือปิดโหมดไว้อาลัยได้อย่างไร =

ไปที่ Settings > โหมดไว้อาลัย แล้วติ๊กหรือเอาติ๊กออกจากช่อง "เปิดใช้งานโหมดไว้อาลัย" แล้วกด Save Changes

= ต้องเตรียมรูปโบว์ดำอย่างไร =

วางไฟล์รูป `black_ribbon_top_right.png` (พื้นหลังโปร่งใส) ในโฟลเดอร์ `/wp-content/plugins/mourning-mode/`

= รองรับ Mobile หรือไม่ =

รองรับครับ ขนาดโบว์จะปรับอัตโนมัติ: Desktop 150px, Tablet 100px, Mobile 80px
